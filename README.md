## Installation

### Custom CSS via Settings

1. Copy the contents of [`theme.css`](theme.css).
2. Open Beeper and go to **Settings** (Gear Icon).
3. Navigate to **Appearance**.
4. Scroll down to the **Custom CSS** section.
5. Paste the copied contents into the text area.
6. The changes should apply immediately (if not, try restarting Beeper).

## Configuration

Since Oxide is designed to be a strictly dark theme, this CSS explicitly overrides both light and dark variables to ensure consistency with the pure dark palette. Make sure you set your theme inside Beeper to Dark so that built-in UI components beyond CSS' reach stay in dark mode.
